The Mondido WooCommerce plugin supports multiple payment methods such as Cards, Invoice, Direct Bank, PayPal and Swish. The subscription service does NOT require WooCommerce Subscriptions, and are free of any additional costs.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin
4. Insert your ID, Secret and API Password that you have in the Mondido Signup (https://www.mondido.com/en)


== Frequently Asked Questions ==

= What is the cost for the plugin? =

None, only the transaction fee from Mondido or any 3:rd party payment provider.
